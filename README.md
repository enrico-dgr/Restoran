## Tech Stack

### Vite + React + TypeScript

- React Router DOM
- Google Fonts
- @fortawesome/react-fontawesome
- React Helmet
- Swiper
- React DatePicker
- Framer Motion ( minimal usage )

### CSS

- SCSS
- BEM
- Responsive
- Minimal size css build through PostCSS and PurgeCSS
